## About Me

I am a **Data Scientist** and **Statistics graduate from Middle East Technical University (METU)** with a strong interest in turning data into practical, real-world solutions.

My work focuses on:

- Building **interactive web applications** for analysis, reporting, and automation using **R Shiny, Streamlit, and Python**
- Developing and optimizing **machine learning models** for applied problems
- Creating **custom web scraping tools** to collect and structure data from various sources
- Deploying and managing applications with **Docker, Nginx, and cloud servers**
- Working with large datasets, conducting **statistical analyses**, and generating actionable insights

I enjoy solving complex problems, learning new tools, and building data products that create measurable impact.
