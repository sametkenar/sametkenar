### Hi there 👋, I am Samet Kenar.

**I am a Data Scientist and Statistics Graduate from Middle East Technical University (METU).**

My main focus is building practical solutions using data. I enjoy tackling complex problems, learning new tools, and turning data into meaningful results. Currently, I'm interested in applied machine learning, automation, and making data products that create real impact.

### 🎯 What I Do:
- **Data Apps & Automation:** Designing interactive web applications for analysis and reporting using R Shiny, Streamlit, and Python.
- **Machine Learning:** Building and optimizing models to solve real-world problems.
- **Data Engineering:** Creating custom web scraping tools for collecting and structuring data from diverse sources.
- **Deployment:** Deploying applications and managing systems using Docker, Nginx, and cloud servers.
